# Интерактивная открытка к 8 марта 🌷

Небольшая интерактивная веб-открытка, созданная к Международному женскому дню.

После нажатия на кнопку **«Получить поздравление»**:
- запускается фоновая музыка;
- текст поздравления появляется с анимацией печати;
- создаётся тёплая праздничная атмосфера.

## Возможности
- Анимация печати текста
- Фоновая музыка
- Плавное интерактивное взаимодействие
- Адаптивный дизайн

## Технологии
- HTML
- CSS
- JavaScript

# Interactive 8 March Greeting Card 🌷

A small interactive web greeting card created for International Women's Day.

When the user clicks the **"Get Greeting"** button:
- background music starts playing;
- the greeting message appears with a typing animation;
- the page creates a warm праздничную atmosphere.

## Features
- Typing text animation
- Background music
- Smooth interactive experience
- Responsive design

## Technologies
- HTML
- CSS
- JavaScript
